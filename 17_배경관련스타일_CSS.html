배경 관련 스타일(background~)

background-color : 배경색을 지정하고자 할 때 사용

같이, 생생하며, 물방아 것이다. 불러 열락의 지혜는 살 인생을 하였으며, 속잎나고, 있는가? 청춘의 할지니, 부패를 착목한는 봄바람이다.

background-clip : 배경을 적용시키고자 하는 범위를 지정할 때

        선택자{
            background-clip : border-box(기본값)/padding-box/content-box;
        }
